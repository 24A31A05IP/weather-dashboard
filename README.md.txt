Weather Dashboards
Project Title

Weather Dashboard Using JavaScript and REST API

Objective

To develop a responsive weather dashboard that fetches and displays real-time weather information for any city using the OpenWeatherMap API. The project demonstrates the use of JavaScript Fetch API, asynchronous programming (async/await), and REST API integration.

Tools & Technologies Used
HTML5
CSS3
JavaScript (ES6)
OpenWeatherMap API
Visual Studio Code
Git & GitHub
Project Overview

The Weather Dashboard is a web application that allows users to search for weather information by entering a city name. The application retrieves real-time weather data from the OpenWeatherMap API and displays details such as temperature, humidity, wind speed, weather condition, and atmospheric pressure.

Features
Search weather by city name
Real-time weather updates
Displays temperature in Celsius
Shows humidity percentage
Displays wind speed
Shows weather description
Error handling for invalid city names
Responsive design for mobile and desktop devices
API Used

OpenWeatherMap API

API Endpoint:

https://api.openweathermap.org/data/2.5/weather

Implementation Steps
Step 1

Create the project structure with HTML, CSS, and JavaScript files.

Step 2

Design the user interface using HTML and CSS.

Step 3

Obtain an API key from OpenWeatherMap.

Step 4

Use JavaScript Fetch API to request weather data.

Step 5

Process the JSON response received from the API.

Step 6

Display weather information dynamically on the webpage.

Step 7

Implement error handling for invalid city names and network issues.

Step 8

Test the application and deploy it to GitHub.

Conclusion

The Weather Dashboard project successfully demonstrates the integration of a REST API with JavaScript to fetch and display real-time weather information. It enhances understanding of web development concepts such as API communication, asynchronous programming, DOM manipulation, and responsive design.
